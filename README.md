# Tesla Text Zapper

A Flet-powered text processing UI that cleans, analyzes, and electrifies your text with Nikola Tesla's essence.

## Features

- **Simple String Processing** - Whitespace cleanup, pawn shop junk removal
- **Advanced Text Processing** - Frequency analysis, pattern detection, energy scoring
- **NLP (No LLM)** - Tokenization, sentiment analysis, complexity rating
- **Tesla Vibes** - Random wisdom quotes from the master himself

## Requirements

- Python 3.8+
- Flet

## Installation

```bash
# Clone/navigate to directory
cd skank_treatment

# Install dependencies
pip install flet

# Run the app
python tesla_text_zapper.py
```

## Usage

1. Launch the app
2. Enter text in the input box
3. Click "ZAP IT!"
4. View cleaned text + NLP analysis in output

## What Gets Cleaned

| Junk Type | Example | Result |
|-----------|---------|--------|
| Sketchy symbols | `★◆●■♠♣` | Removed |
| Excessive punctuation | `!!!` `???` `$$$` | Single char |
| ALL CAPS SCREAMING | `BUY NOW` | `Buy Now` |
| Extra whitespace | `hello   world` | `hello world` |

## Output Breakdown

```
ENERGY: Word diversity percentage
PATTERNS: Repeated words detected
VIBE: Positive/Negative/Neutral charge
COMPLEXITY: High/Low voltage based on avg word length
```
